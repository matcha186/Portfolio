Game
ほしぞらアート

ABOUT


『ほしぞらアート』 は、「自分だけの星座を作ろう」 をコンセプトに、Scratchで制作したゲーム。
夜空にランダムに現れる星を自由につなぎ、自分だけのオリジナルな星座やアートを作ることができる。
シンプルながらも創造力を刺激するゲームデザインになっており、完成した作品を眺めながら、まるで本当に新しい星座を発見したような感覚を味わえる。
本作のゲームシステム、設計、BGM制作まで、すべてを一人で手がけた。
幻想的な雰囲気を大切にし、プレイヤーが星空を眺めるように楽しめる演出を取り入れている。

DETAILS

使用技術・ツール

  • 言語: Scratch
  • 使用ツール: Scratch

開発概要

大学時代のアルバイト先のプログラミング教室で、子供たちにリストの使い方をわかりやすく伝えるために制作。


開発期間

約2週間


開発体制

1人で開発


こだわったポイント

  • ボタンのホバー・クリック時の挙動を工夫し、操作感を向上
  • 作成した星座を保存し、後から見返せる機能を実装

課題と対応

  • 星座の保存機能の実装に苦戦
  • 各星の座標をリストに保存し、奇数要素をx座標、偶数要素をy座標として管理することで対応

成果と学び

  • Scratchのリスト機能を習得
  • リスト（配列）の概念がPythonやJavaScriptにも応用できることを実感
