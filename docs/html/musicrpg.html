Game
Music RPG

ABOUT


『Music RPG』 は、「音楽の楽しさをゲームを通じて知ってもらう」 をコンセプトに制作した、謎解き要素を含むRPGゲーム。
新たなキャラクターが仲間になるたびに、ゲーム内のBGMに新しい楽器が追加され、音楽がより豊かになっていく仕組み。
物語とともに音楽が進化し、プレイヤー自身が楽曲を完成させていくような体験ができるのが特徴。
本作のゲームシステム、設計、BGM制作まで、すべてを一人で手がけた。
ゲームデザインだけでなく、音楽の組み込みにもこだわり、プレイヤーが音の変化を直感的に楽しめるよう工夫している。
また本作は、大学の卒業制作で制作した作品であり、学科賞を受賞している。

DETAILS

使用技術・ツール

  • 言語: C#
  • 使用ツール: Unity, VSCode

開発概要

音楽の楽しさをゲームを通じて知ってもらうことを目的とし、プレイヤーが謎解きをしながらキャラクターを仲間にしていくRPGゲームを制作した。


開発期間

約半年


開発体制

1人で開発（BGMも制作）


こだわったポイント

  • キャラクターのアニメーションや画面の動きにこだわり、市販のRPGゲームに近づけた
  • エリア移動時のフェードアウト処理を導入し、安っぽさを感じさせないよう工夫
  • GitHubを活用し、バージョン管理を行いながら効率的に作業を進めた

課題と対応

  • キャラクターを仲間に加えるごとにBGMに楽器を追加する仕組みの実装に苦戦
  • 最終的に、すべてのBGMパターンを用意し、キャラクター加入フラグに応じて適切なBGMを再生する方式を採用

成果と学び

  • Unityの基礎技術の習得と、C#の知識・コーディングスキルを向上
  • 知人に限定公開してプレイしてもらい、バグを発見することでデバッグ方法を学んだ
