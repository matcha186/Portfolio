ABOUT


洋服の上下の色合わせを試せるWebアプリ。
ユーザーはまず、「フェミニン」「カジュアル」のスタイルを選択する。
その後、トップスの色をカラーパレットから選ぶと、スタイルに合ったボトムスのおすすめの色が自動で表示される。
ユーザーはその中から好みの色を選ぶことで、簡単にコーディネートのシミュレーションを行うことができる。
このアプリを活用することで、洋服の色合わせに悩む時間を短縮でき、新しい組み合わせの発見にもつながる。

DETAILS

使用技術・ツール

  • 言語: HTML, CSS, JavaScript
  • 使用ツール: VSCode, Chrome

開発概要

本サイトは、3か月にわたる他社のIT研修プログラムの最後に取り組んだチーム制作課題として作成した。
HTML, CSS, JavaScriptを使用することが必須であり、チームメンバーが自分以外女性だったことから、女性向けのファッションコーディネート支援サイトを企画した。


開発期間

約3週間


開発体制

4人チームで開発を行い、プロジェクトマネージャー2名、デザイナー1名、プログラマー1名に役割分担した。
自分はプログラマーとして、メインページのプログラムをすべて担当した。
また、メインページ（パレット画面）のBGMは自分が作成した。


こだわったポイント

  • ボタンのホバー時に浮き出るエフェクトを実装
  • 服の色を変更した際にエフェクトが出現する仕様を実装
  • UIの視認性や操作性を意識したデザイン設計

課題と対応

  • 服の色の組み合わせをDBに登録して呼び出す機能を試みたが、実装が難しくフロントエンドのみで作成
  • AWSを利用し公開を目指したが、時間の都合でローカル環境での実装に留まる

成果と学び

  • Webサイト制作におけるUI要素の作成・配置の設計を習得
  • チーム開発におけるPMとのプロジェクト進行やテスト仕様書の作成を経験
