ヒモ勇者

ABOUT


かつて裕福な家で何不自由なく暮らしていた主人公の男・アレックス。しかし、ある日突然、実家が倒産し、すべてを失ってしまう――。働く気などさらさらない彼が選んだ道は、“ヒモ”になること。
そこで彼は一世一代の大博打に出る。伝説の勇者を装い、お城の舞踏会へと潜り込み、姫との接触を狙うのだった。
王国には二人の姫がいることを知っていた彼は、どちらかのヒモになること、さらには両方のヒモになることすら視野に入れていた。
しかし、計画を進めるうちに、彼の前に謎の男が現れる――。
果たして主人公は誰かの「ヒモ」になることができるのか？ それとも、予想もしなかった真実へと辿り着くのか？
嘘と策略、恋と陰謀が交錯する恋愛シミュレーションWebノベルゲーム。すべてのルートを攻略し、真のトゥルーエンドを目指せ！

DETAILS

使用技術・ツール

  • 言語: HTML, CSS, JavaScript
  • 使用ツール: VSCode, Chrome

開発概要

研修期間中に自主学習の時間が設けられた際、同期5人で「ノベルゲームを作ろう」となり、制作に取り組んだ。


開発期間

約3週間


開発体制

5人チームで開発を行い、自分含めプログラマー2人でプログラムを担当した。他の3人はプロジェクトマネージャー2人、デザイナー1人として携わった。


こだわったポイント

  • もう1人のプログラマーと役割分担し、効率的に開発を進めた
  • ゲーム完成後にエンド一覧や早送り機能を追加し、ユーザーがより楽しめる仕様にした

課題と対応

  • 開発初期には予定していなかった早送り機能の追加に苦労した
  • 通常の「クリックで次へ進む」処理をスキップする仕組みを後付けで実装
  • 今後の開発では、企画段階で拡張機能も考慮して設計する重要性を実感

成果と学び

  • Webサイト制作におけるUI要素の作り方や配置の考え方を習得
  • チーム制作でのプロジェクト進行方法やテスト仕様書の作成方法を経験
