Game
White Quest

ABOUT


『White Quest』 は、アナログダイスゲームをデジタル化した、戦略性のある対戦ゲーム。
本作は、あるゲーム実況グループの雑誌に掲載されていたダイスゲームをもとに制作した。
6人のキャラクターが登場し、それぞれが持つダイスの目によって多彩な技や効果を発動。
相手のHPを0にすれば勝利というシンプルなルールながらも、キャラクターの組み合わせ次第で戦術が大きく変わるため、何度でも楽しめる。
本作のUI、プログラムはすべて一人で手がけた。

DETAILS

使用技術・ツール

  • 言語: HTML, CSS, JavaScript
  • 使用ツール: VSCode, Chrome

開発概要

アナログでこのゲームをプレイしているうちに、「自分の力量でデジタル化できそう」と感じ、JavaScriptの理解を深める目的も兼ねて制作した。


開発期間

約1か月


開発体制

1人で開発


こだわったポイント

  • ダメージを与えた際に画面が揺れる仕様を実装
  • 特定の技にエフェクトを追加し、視覚的に楽しめる演出を強化

課題と対応

  • 相手のターンをスキップする技や、ダメージを軽減する技など、多くの種類の技を一つずつ作成するのに苦労
  • 技を追加するたびにバグが発生し、その都度デバッグを繰り返しながら修正

成果と学び

  • プレイヤーが楽しめる演出や、操作しやすいUI設計を考える力が向上
  • 自分のターンと相手のターン、先攻と後攻など、複雑な場合分け処理を実装する力を習得
  • キャラクターの技を実装するたびに新しいフラグ変数を作成していたが、最初に変数を整理しておけばもっと効率よく開発できたと感じた
  • プログラムを実装する前に仕様書を作成する重要性を実感
