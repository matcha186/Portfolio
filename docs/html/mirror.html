Game
Mirror Drop

ABOUT


『Mirror Drop』 は、「反転する世界でどこまで生き延びられるか」 をテーマにしたアクションゲーム。
落ちてくるトゲを避けながらスコアを伸ばしていくシンプルなルールだが、本作の特徴は操作が逆である。
そして、プレイヤーの動きも鏡合わせになっているため、直感的な操作が通用せず、頭がこんがらがるような感覚を味わえる。
本作のゲームシステム、設計、BGM制作まで、すべてを一人で手がけた。
シンプルながらも奥深い、"脳がバグる" 体験をぜひ楽しんでほしい。

DETAILS

使用技術・ツール

  • 言語: Scratch
  • 使用ツール: Scratch

開発概要

自身のScratchの理解度を深めるために自主的に作成した。特に、定義ブロック（関数）の学習を目的としている。


開発期間

約1週間


開発体制

1人で開発（BGMも制作）


こだわったポイント

  • 定義ブロック（関数）を活用することで、よりスマートにコードを記述
  • 引数の値を変更して同じ定義ブロックを何度も呼び出し、処理を効率化

課題と対応

  • 定義ブロック（関数）の概念を知らない状態で開発を進めたため、試行錯誤が必要だった
  • しかし、使い方を理解するとコードを簡略化できることがわかり、積極的に活用

成果と学び

  • Scratchの定義ブロック（関数）の活用方法を習得
  • 関数の概念を学び、本格的なプログラミング（PythonやJavaScriptなど）にも応用できる知識を得た
